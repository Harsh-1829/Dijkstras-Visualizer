About Dijkstra's Algorithm

Dijkstra's algorithm finds the shortest path between nodes in a graph with non-negative edge weights.

How to Use:

  • Presets: Choose from pre-built graph examples.
  • Matrix: Create a graph using an adjacency matrix.
  • Edge List: Define edges manually (one per line: `from to weight`).
  • Drag Nodes: Click and drag nodes on the canvas to reposition them.

Controls:

  • Start: Begin the algorithm animation.
  • Pause/Resume: Control animation playback.
  • Step: When paused, move through the algorithm one step at a time.
  • Reset: Return the graph to its initial state.
